Private CA onboarding

Trust your homelab HTTPS

Install only the public Sovereign Homelab root certificate. This portal never contains a CA private key, application password, or API token.

Verify first: compare the SHA-256 value in fingerprint.txt with the value shown in the Proxmox root-only CA inventory.

Windows and Firefox

  1. Download the certificate and installer.
  2. Open PowerShell as Administrator.
  3. Run the installer from the download directory.
  4. Restart Firefox and test the four critical aliases.
Windows installer Root certificate

iPhone and iPad

  1. Open this page while connected to LAN or VPN.
  2. Install the downloaded profile in Settings.
  3. Go to Settings > General > About > Certificate Trust Settings.
  4. Enable full trust for the Sovereign Homelab root.
Apple profile

macOS

  1. Download the root certificate.
  2. Import it into the System keychain with Keychain Access.
  3. Open its Trust section and set SSL trust to Always Trust.
  4. Restart the browser.
Root certificate

Android

  1. Download the root certificate.
  2. Use Security > Encryption and credentials > Install a certificate > CA certificate.
  3. Confirm the fingerprint before accepting.
  4. Some Android apps ignore user-installed CAs; browsers normally support them.
